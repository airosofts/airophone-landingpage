# AiroPhone Landing Page

Marketing landing page for AiroPhone - SMS & Voice Campaign Platform

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the landing page.

## Deployment

This landing page should be deployed to **airophone.airosofts.com**

The main application is deployed separately to **ap.airosofts.com**

## Features

- Responsive design
- Business inquiry form
- Pricing tiers display
- Feature showcase
- Same theme as main application
